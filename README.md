# Intension of 1962 Game

Tools and Languages
- C++
- SFML C++ Library
- Visual Studio 2022

**Main Requirement**
- Player have gun in default, health of 100 and velocity of 5
- Player can collect first-aid, bullet, and microfilm item
- Player cannot wander outside map
- Enemy have gun in default, idle (not alert) state, health of 100 and velocity of 5
- Enemy elevate state to alert state when player is shooting at them
- Enemy have tracking player coordinates system when follow player in alert mode
- Microfilm must be generated in random room
- Items in each room must be generated randomly every 30 seconds
- Game have statistic calculate system in each round
